HTML

HTML, que significa HyperText Markup Language, es el lenguaje estándar utilizado para crear páginas web. Es un lenguaje de marcado, lo que significa que utiliza etiquetas para estructurar y describir el contenido de una página web.

CSS

CSS (Cascading Style Sheets) es el lenguaje que se utiliza para describir la presentación de un documento escrito en HTML. Mientras que HTML estructura el contenido de la página, CSS controla su apariencia, proporcionando estilos para elementos como colores, fuentes, márgenes, espaciados y el diseño general de la página.

Last updated 3 mins ago

JavaScript

JavaScript es un lenguaje de programación de alto nivel y dinámico, esencial para el desarrollo web. Permite a los desarrolladores crear contenido interactivo y dinámico en las páginas web.

React

breve definicion de react React es una biblioteca de JavaScript desarrollada por Facebook, utilizada para construir interfaces de usuario, especialmente aplicaciones web de una sola página (Single Page Applications o SPAs). Se enfoca en crear componentes reutilizables que manejan su propio estado y generan interfaces dinámicas y eficientes.

NodeJS

breve definicion de node js Node.js es un entorno de ejecución de JavaScript en el servidor. Permite ejecutar código JavaScript fuera de un navegador, utilizando el motor V8 de Google Chrome.

Bootstrap

Bootstrap es un framework front-end de código abierto desarrollado por Twitter, que se utiliza para diseñar sitios y aplicaciones web responsive y con un estilo moderno. Ofrece una colección de herramientas que incluyen componentes HTML, CSS y JavaScript predefinidos y fácilmente personalizables.
